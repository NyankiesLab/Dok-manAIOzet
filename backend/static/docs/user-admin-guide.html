👤 Kullanıcı Kılavuzu

📝 Kayıt Olma

Adım 1: Kayıt Formunu Doldurun

  1. Ana sayfada "Kayıt Ol" butonuna tıklayın
  2. E-posta adresinizi girin
  3. Kullanıcı adınızı belirleyin
  4. Güçlü bir şifre oluşturun
  5. Şifrenizi tekrar girin
  6. "Kayıt Ol" butonuna tıklayın

Güvenlik İpuçları

  • En az 8 karakterli şifre kullanın
  • Büyük/küçük harf, rakam ve özel karakter içersin
  • Kişisel bilgilerinizi şifrede kullanmayın
  • E-posta adresinizi doğru girdiğinizden emin olun

🔐 Giriş Yapma

Giriş İşlemi

  1. Ana sayfada "Giriş" butonuna tıklayın
  2. E-posta adresinizi girin
  3. Şifrenizi girin
  4. "Giriş Yap" butonuna tıklayın
  5. Başarılı giriş sonrası dashboard'a yönlendirilirsiniz

Sorun Giderme

  • Şifrenizi unuttuysanız "Şifremi Unuttum" linkini kullanın
  • E-posta adresinizi kontrol edin
  • Caps Lock tuşunun açık olmadığından emin olun
  • Tarayıcı önbelleğini temizleyin

📤 Doküman Yükleme

Desteklenen Formatlar

  • PDF: .pdf dosyaları
  • Word: .docx, .doc dosyaları
  • Metin: .txt dosyaları

Not: Maksimum dosya boyutu 10MB'dır.

Yükleme Adımları

  1. Dashboard'da "Doküman Yükle" butonuna tıklayın
  2. Dosya seçme alanına tıklayın
  3. Yüklemek istediğiniz dosyayı seçin
  4. Dosya adını ve açıklamasını girin (opsiyonel)
  5. "Yükle" butonuna tıklayın
  6. Yükleme tamamlandığında onay mesajı alırsınız

🔍 Doküman Arama

Arama Türleri

  • Basit Arama: Anahtar kelime ile arama
  • Gelişmiş Arama: Filtreler ile arama
  • Tam Metin Arama: İçerikte arama
  • Tarih Filtresi: Tarih aralığı ile arama

Arama İpuçları

  • Tırnak işareti kullanarak tam eşleşme arayın
  • AND, OR operatörlerini kullanın
  • Dosya türü filtresini kullanın
  • Tarih aralığı belirleyin

🤖 AI Özetleme

Özetleme Özellikleri

  • Otomatik özetleme
  • Anahtar kelime çıkarımı
  • İçerik kategorilendirme
  • Çoklu dil desteği

Kullanım Adımları

  1. Dokümanınızı seçin
  2. "AI Özetle" butonuna tıklayın
  3. Özet uzunluğunu seçin
  4. Özetleme işlemini bekleyin
  5. Sonuçları görüntüleyin ve kaydedin

⚙️ Admin Kılavuzu

📊 Admin Paneli

Panel Özellikleri

  • Sistem istatistikleri
  • Kullanıcı aktivite raporları
  • Doküman yükleme analizleri
  • Sistem performans metrikleri

Erişim

  1. Admin hesabıyla giriş yapın
  2. Dashboard'da "Admin Paneli" linkine tıklayın
  3. İstatistikleri görüntüleyin
  4. Raporları indirin

👥 Kullanıcı Yönetimi

Kullanıcı İşlemleri

  • Kullanıcı listesini görüntüleme
  • Kullanıcı hesaplarını askıya alma
  • Kullanıcı rollerini değiştirme
  • Kullanıcı aktivite geçmişini görüntüleme

Güvenlik Önlemleri

  • Şüpheli aktiviteleri izleme
  • Başarısız giriş denemelerini takip etme
  • Kullanıcı hesaplarını kilitleme
  • Otomatik uyarı sistemi

📈 Sistem İzleme

İzleme Metrikleri

  • CPU ve RAM kullanımı
  • Disk alanı durumu
  • Veritabanı performansı
  • API yanıt süreleri

Uyarı Sistemi

  • Yüksek sistem kullanımı uyarıları
  • Disk alanı kritik seviye uyarıları
  • Hata oranı uyarıları
  • E-posta bildirimleri

💾 Yedekleme ve Geri Yükleme

Otomatik Yedekleme

  • Günlük veritabanı yedekleme
  • Haftalık tam sistem yedekleme
  • Bulut depolama entegrasyonu
  • Yedekleme doğrulama

Geri Yükleme İşlemleri

  1. Yedek dosyasını seçin
  2. Geri yükleme noktasını belirleyin
  3. Geri yükleme işlemini başlatın
  4. Doğrulama testleri yapın

🔒 Güvenlik Ayarları

Güvenlik Konfigürasyonu

  • JWT token süre ayarları
  • Şifre politikası yapılandırması
  • IP kısıtlamaları
  • Rate limiting ayarları

Güvenlik İzleme

  • Başarısız giriş denemeleri
  • Şüpheli IP adresleri
  • Anormal kullanıcı aktiviteleri
  • Sistem güvenlik logları
