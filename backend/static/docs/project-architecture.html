Proje Mimarisi ve API Dokümantasyonu

Doküman Forum'un teknik mimarisi, API endpoint'leri ve sistem yapısı hakkında detaylı bilgiler.

Sistem Mimarisi

Backend

FastAPI + Python

Veritabanı

SQLite + SQLAlchemy

AI

Google Gemini API

Güvenlik

JWT + bcrypt

API Endpoint'leri

POST /api/auth/register

Açıklama: Yeni kullanıcı kaydı

{ "email": "user@example.com", "username": "username", "password": "password", "full_name": "Full Name" }

POST /api/auth/login

Açıklama: Kullanıcı girişi

{ "email": "user@example.com", "password": "password" }

POST /api/documents/

Açıklama: Doküman yükleme

FormData: { "file": "document.pdf", "title": "Document Title" }

GET /api/search/?query=search_term

Açıklama: Doküman arama

Query Parameters: - query: Arama terimi - file_type: Dosya türü filtresi

POST /api/summary/{id}/generate

Açıklama: AI ile doküman özetleme

Response: { "summary": "Özet metni", "keywords": "anahtar, kelimeler" }

Veritabanı Şeması

Users Tablosu

Alan Tip Açıklama
id Integer Birincil anahtar
email String Benzersiz email adresi
username String Benzersiz kullanıcı adı
hashed_password String Şifrelenmiş parola
full_name String Tam ad

Documents Tablosu

Alan Tip Açıklama
id Integer Birincil anahtar
title String Doküman başlığı
filename String Dosya adı
file_path String Dosya yolu
file_size Integer Dosya boyutu (bytes)
file_type String Dosya türü
content Text Doküman içeriği
summary Text AI özeti
keywords String Anahtar kelimeler
user_id Integer Kullanıcı referansı

Güvenlik

Kimlik Doğrulama

  • JWT (JSON Web Token) tabanlı kimlik doğrulama
  • bcrypt ile şifre hashleme
  • Token tabanlı oturum yönetimi
  • Güvenli API endpoint koruması

Dosya Güvenliği

  • Dosya türü doğrulama
  • Dosya boyutu sınırlaması
  • Güvenli dosya yükleme yolu
  • Kullanıcı bazlı dosya erişimi

Deployment

Gereksinimler

Python 3.8+ FastAPI Uvicorn SQLAlchemy Google Generative AI

Kurulum

pip install -r requirements.txt python main.py

Environment Variables

DATABASE_URL=sqlite:///./document_db.sqlite SECRET_KEY=your-secret-key GEMINI_API_KEY=your-gemini-api-key
