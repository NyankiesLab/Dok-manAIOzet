🏠 Sistem Genel Bakış

Doküman Yönetim Sistemi Mimarisi

Bu sistem, modern web teknolojileri kullanılarak geliştirilmiş kapsamlı bir doküman yönetim platformudur. FastAPI backend'i, SQLAlchemy ORM'i ve Google Gemini AI entegrasyonu ile güçlendirilmiştir.

Backend Teknolojileri

  • • FastAPI (ASGI Framework)
  • • SQLAlchemy (ORM)
  • • SQLite/PostgreSQL
  • • JWT Authentication
  • • Google Gemini AI

Frontend Teknolojileri

  • • HTML5 + JavaScript
  • • Tailwind CSS
  • • Axios (HTTP Client)
  • • Responsive Design

🔐 Kimlik Doğrulama Modülü

Dosya Yapısı

models/user.py # Kullanıcı modeli services/auth_service.py # Kimlik doğrulama servisi api/routes/auth.py # Auth API endpoint'leri

Özellikler

  • JWT token tabanlı kimlik doğrulama
  • Kullanıcı kayıt ve giriş işlemleri
  • Şifre hash'leme (bcrypt)
  • Token yenileme ve doğrulama
  • Kullanıcı profil yönetimi

API Endpoint'leri

POST /api/auth/register - Kullanıcı kaydı
POST /api/auth/login - Kullanıcı girişi
GET /api/auth/me - Mevcut kullanıcı bilgileri

📄 Doküman Yönetimi Modülü

Dosya Yapısı

models/document.py # Doküman modeli services/document_service.py # Doküman işlemleri api/routes/documents.py # Doküman API endpoint'leri

Desteklenen Dosya Türleri

PDF - PyPDF2 ile işleme
DOC/DOCX - python-docx ile işleme
TXT - Doğrudan metin işleme

API Endpoint'leri

POST /api/documents/ - Doküman yükleme
GET /api/documents/ - Doküman listesi
GET /api/documents/{id} - Doküman detayı
DELETE /api/documents/{id} - Doküman silme

🔍 Arama Modülü

Dosya Yapısı

api/routes/search.py # Arama API endpoint'leri services/ai_service.py # AI destekli arama

Arama Türleri

Basit Metin Arama

Kelime eşleşmesi ile hızlı arama

AI Destekli Arama

Gemini API ile akıllı arama

API Endpoint'leri

GET /api/search/ - Temel arama
GET /api/search/ai-search - AI destekli arama
GET /api/search/advanced-search - Gelişmiş arama

🤖 AI Servisi

Dosya Yapısı

services/ai_service.py # AI servisi app/config.py # Gemini API konfigürasyonu

AI Özellikleri

📝 Otomatik Özetleme

Doküman içeriğinden AI ile özet çıkarma

🔑 Anahtar Kelime Çıkarma

Önemli kelimeleri otomatik tespit

Konfigürasyon

GEMINI_API_KEY = "your-api-key" GEMINI_MODEL = "gemini-1.5-pro"

📝 Özetleme Modülü

Dosya Yapısı

api/routes/summary.py # Özetleme API endpoint'leri services/ai_service.py # AI özetleme servisi

API Endpoint'leri

POST /api/summary/{id}/generate - Özet oluşturma

Özellikler

  • Detaylı doküman özeti (300-400 kelime)
  • Anahtar kelime çıkarma (15 kelime)
  • Türkçe dil desteği
  • Fallback mekanizması (AI olmadığında)

🗄️ Veritabanı Modülü

Dosya Yapısı

app/database.py # Veritabanı konfigürasyonu models/user.py # Kullanıcı tablosu models/document.py # Doküman tablosu

Veritabanı Şeması

Users Tablosu

  • • id (Primary Key)
  • • email (Unique)
  • • username
  • • full_name
  • • hashed_password
  • • is_active
  • • created_at

Documents Tablosu

  • • id (Primary Key)
  • • title
  • • filename
  • • file_path
  • • file_size
  • • file_type
  • • content
  • • summary
  • • keywords
  • • user_id (Foreign Key)

⚙️ Konfigürasyon Modülü

Dosya Yapısı

app/config.py # Ana konfigürasyon dosyası .env # Ortam değişkenleri (opsiyonel)

Konfigürasyon Parametreleri

DATABASE_URL - Veritabanı bağlantı URL'i
SECRET_KEY - JWT imzalama anahtarı
GEMINI_API_KEY - Google Gemini API anahtarı
ALLOWED_ORIGINS - CORS izin verilen domain'ler
UPLOAD_DIR - Dosya yükleme dizini
